# Catching Bananas Game (Scratch)

## Description
A simple interactive game built using Scratch where the player catches falling bananas to score points.

## Features
- Score counter
- Falling object animation
- Player-controlled movement
- Game over condition

## What I Learned
- Basic game logic
- Event handling
- Loops and conditions
- Sprite interaction
